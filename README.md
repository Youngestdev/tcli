## Trieve

Trieve is a simple command line tool that gets the details of a Twitter user from the supplied username

## Installation

### Via NPM

```bash
>  npm i trieve 
```

### Via Yarn

```bash
> yarn add trieve
```

## Using trieve

```bash
trieve --username "username" 
```

or with **npx**

```bash

> npx trieve --username unicodeveloper

```
